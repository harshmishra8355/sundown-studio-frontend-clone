# 🎨 Sundown Studio Clone

A modern and responsive frontend clone of the Sundown Studio website built using HTML, CSS, JavaScript, GSAP, and Locomotive Scroll.

## 🚀 Features

* Responsive Landing Page
* Smooth Scrolling using Locomotive Scroll
* GSAP Animations
* Interactive Hover Effects
* Dynamic Image Preview on Hover
* Modern UI Design
* Video Background Section
* Custom Scroll Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP
* ScrollTrigger
* Locomotive Scroll

## 📚 What I Learned

* Implementing smooth scrolling experiences
* Creating advanced animations with GSAP
* Working with ScrollTrigger
* Building responsive layouts
* Improving UI/UX design skills
* Handling DOM events and hover interactions

## 🎯 Project Goal

The goal of this project was to improve my frontend development skills by recreating a modern agency website and implementing professional animations and interactions.
